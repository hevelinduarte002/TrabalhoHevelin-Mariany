# TrabalhoHevelin-Mariany

Relatório do Projeto – Jogo “Acerte o Número” em C#

O presente trabalho teve como objetivo o desenvolvimento de um jogo de adivinhação utilizando a linguagem de programação C#, com foco na aplicação de estruturas de decisão, repetição, funções e validação de dados. O projeto foi realizado em grupo, no qual a lógica do sistema foi implementada por mim, enquanto a parte de documentação ficou sob responsabilidade da colega Mariany. A proposta seguiu as orientações fornecidas, visando reforçar conceitos fundamentais da programação.

O jogo consiste em desafiar o usuário a adivinhar um número secreto gerado aleatoriamente pelo programa, dentro do intervalo de 1 a 20. A cada rodada, o sistema utiliza a classe Random para definir esse número, garantindo variação a cada execução. O jogador deve inserir um valor inteiro e o programa exibe o número informado, conforme solicitado na atividade, iniciando assim o processo de verificação.

Para o funcionamento do jogo, foram utilizadas estruturas condicionais (if e else) para comparar o número digitado com o número secreto. Quando o jogador acerta, é exibida a mensagem de sucesso “PARABÉNS, VOCÊ ACERTOU!”, juntamente com a quantidade de tentativas realizadas. Caso contrário, o programa informa que o jogador não acertou e fornece dicas indicando se o número secreto é maior ou menor que o valor digitado, permitindo uma melhor tomada de decisão nas próximas tentativas.

Além disso, foi implementado um laço de repetição (while) que mantém o jogo em execução até que o jogador acerte o número ou decida sair digitando 0. O sistema também oferece a opção de reiniciar o jogo ao final de cada partida, tornando a aplicação mais interativa. Outro ponto importante é a utilização de uma função específica para validar a entrada do usuário, garantindo que apenas valores numéricos dentro do intervalo permitido sejam aceitos, evitando erros durante a execução.

Por fim, o projeto atendeu aos requisitos propostos, apresentando um código organizado, funcional e de fácil compreensão. A atividade contribuiu significativamente para o aprendizado prático dos conceitos abordados em sala, além de incentivar o trabalho em equipe e a divisão de responsabilidades. O resultado final foi um jogo simples, porém eficiente, demonstrando a correta aplicação das estruturas de controle e boas práticas de programação em C#.
